# Go Auth Service (Gin + GORM)

A simple authentication backend built with **Golang**, **Gin**, and **GORM**.
This project demonstrates core backend concepts like JWT authentication, password hashing, and clean structure.

---

## 🚀 Features

* User Registration
* User Login
* JWT Authentication
* Password Hashing (bcrypt)
* Protected Routes (middleware)
* PostgreSQL + GORM

---

## 🛠 Tech Stack

* Go (Golang)
* Gin (HTTP framework)
* GORM (ORM)
* PostgreSQL
* JWT (Authentication)
